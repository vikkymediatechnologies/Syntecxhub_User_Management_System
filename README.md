# User Management System API

## Features
- CRUD operations (Create, Read, Update, Delete)
- MongoDB database integration
- Mongoose schema
- Basic Authentication
- Tested with Postman

## Technologies Used
- Node.js
- Express.js
- MongoDB
- Mongoose

## API Endpoints

| Method | Endpoint |
|--------|---------|
| POST | /api/users |
| GET | /api/users |
| GET | /api/users/:id |
| PUT | /api/users/:id |
| DELETE | /api/users/:id |
